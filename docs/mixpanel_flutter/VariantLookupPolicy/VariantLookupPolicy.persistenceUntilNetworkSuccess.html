Serve persisted variants immediately on init (within persistenceTtl), then refresh from the network in the background.

Web: not yet supported by the Mixpanel JS SDK at the time of this release. On web this policy is silently treated as networkOnly until JS SDK support ships. Check the Mixpanel JS docs for availability.
