source property

MixpanelFlagVariantSource source
final

Where this variant was sourced from. Non-null on every variant the SDK returns. Variants the developer constructs default to FallbackSource, since the only reason to build one is to pass it as a fallback.

Use is checks to distinguish:

final src = variant.source;
if (src is PersistenceSource) {
  print('persisted at ${src.persistedAt}');
} else if (src is NetworkSource) {
  print('fresh from /flags/');
} else if (src is FallbackSource) {
  print('developer-supplied fallback');
}
