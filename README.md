# mixpanel-flutter

Monorepo for the Mixpanel Flutter SDK family.

## Packages

| Package | Description |
|---|---|
| [`mixpanel_flutter`](packages/mixpanel_flutter) | Official Flutter SDK for Mixpanel Analytics |
| [`mixpanel_flutter_session_replay`](packages/mixpanel_flutter_session_replay) | Official Flutter Session Replay SDK for Mixpanel |
| [`mixpanel_flutter_common`](packages/mixpanel_flutter_common) | Shared common library for the Mixpanel Flutter SDK family |

See each package's README for installation and usage.
